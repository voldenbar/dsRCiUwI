# 项目简介

基于YOLOv8v10v11深度学习技术的X光安检危险品图像识别系统，旨在提升安检效率与准确性。系统包括Python源码、用户界面、数据集及训练代码，具备物体检测、图像识别功能。技术特点在于运用了最新的深度学习算法，实现了对X光图像中潜在危险品的快速准确识别，有效支持安检人员作业，保障公共安全。


## 核心功能
- 基于YOLOv8、YOLOv10和YOLOv11深度学习算法的X光安检危险品图像识别
- 物体检测与图像识别功能
- 提供完整的Python源代码实现
- 包含用户界面(UI)组件
- 提供专用数据集用于模型训练
- 包含训练代码支持模型优化

## 适用场景
- 机场、车站等公共场所的安全检查
- 行李包裹中的危险品识别
- 安检人员辅助工具
- 公共安全保障系统

## 技术特点
- 采用最新的深度学习算法提升识别准确性
- 支持快速处理X光图像
- 专为安检场景优化的模型架构
- 完整的开发环境与部署方案
